# WordPress React starter theme

Work in progress.

Build with [Inertia.js](https://inertiajs.com) and [server-side adapter for WordPress](https://github.com/boxybird/inertia-wordpress/).

## Installation

composer install<br />
npm install<br />

## Build 

Mix using [Laravel mix](https://laravel-mix.com/)

npx mix<br />
npx mix watch<br />
npx mix -p<br />
